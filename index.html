Server Docs

Update & Upgrade

Update package lists

Command 
sudo apt update

Upgrade packages

Command 
sudo apt upgrade

User Setup

Delete default 'ubuntu' user 

sudo deluser ubuntu

Create new 'ubuntu' user 

sudo adduser --gecos "" ubuntu

Add user to sudo group 

sudo usermod -aG sudo ubuntu

SSH Configuration

Generate SSH key (Local) 

ssh-keygen -t rsa -b 4096 -C "your_email@example.com"

Setup Authorized Keys (Server) 

sudo mkdir -p /home/ubuntu/.ssh
sudo chown ubuntu:ubuntu /home/ubuntu/.ssh
sudo chmod 700 /home/ubuntu/.ssh
sudo touch /home/ubuntu/.ssh/authorized_keys
sudo chown ubuntu:ubuntu /home/ubuntu/.ssh/authorized_keys
sudo chmod 600 /home/ubuntu/.ssh/authorized_keys

Firewall (UFW)

Reset & Defaults 

sudo ufw reset
sudo ufw default deny incoming
sudo ufw default allow outgoing

Allow Services 

# SSH
sudo ufw allow 22/tcp limit 10/minute

# HTTP/HTTPS
sudo ufw allow 80/tcp
sudo ufw allow 443/tcp

Enable UFW 

sudo ufw enable

Timezone

Set to Asia/Kolkata 

sudo timedatectl set-timezone Asia/Kolkata

Security Hardening

Disable Root Login 

sudo sed -i 's/^PermitRootLogin.*/PermitRootLogin no/' /etc/ssh/sshd_config

Disable Password Authentication 

sudo sed -i 's/^PasswordAuthentication.*/PasswordAuthentication no/' /etc/ssh/sshd_config

Reboot 

sudo reboot

Nginx Setup

Install Nginx & Certbot 

sudo apt install nginx
sudo apt install certbot python3-certbot-nginx

Docker Setup

Install Docker 

sudo apt install docker.io

Add user to docker group 

sudo usermod -aG docker ubuntu

Django Project

Create build.sh 

rm -rf _code
git clone -b master GIT_REPO_URL _code
sed -i "s/DEBUG = True/DEBUG = False/" _code/DJANGO_PROJECT_NAME/settings.py
docker compose up --build -d
Make executable chmod +x build.sh

Create Dockerfile 

FROM python:VERSION
WORKDIR /app
ENV PYTHONUNBUFFERED 1
COPY _code/requirements.txt .
RUN pip install -r requirements.txt
COPY _code/ /app
CMD gunicorn --bind 0.0.0.0:8000 DJANGO_PROJECT_NAME.wsgi:application

Create docker-compose.yml 

services:
    app:
        build: .
        ports:
            - "127.0.0.1:10001:8000"
        volumes:
            - ./media:/app/media
            - ./db.sqlite3:/app/db.sqlite3
        restart: always

Nginx Config 

server {
    server_name domain_name.in;

    location /media/ {
        alias /app/media/;
    }

    location / {
        proxy_pass http://127.0.0.1:10001;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }
}
Edit sudo nano /etc/nginx/sites-available/domain_name.conf
Link sudo ln -s /etc/nginx/sites-available/domain_name.conf /etc/nginx/sites-enabled/

SSL Certificate

Obtain SSL via Certbot 

sudo certbot --nginx -d domain_name.in

Congratulations! Your server and Django project are ready.
